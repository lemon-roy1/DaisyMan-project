what is dom in javascript ?

DOM stands for "Document Object Model." In JavaScript, it refers to a programming interface for web documents. It represents the structure of an HTML or XML document as a tree-like structure, allowing developers to access and manipulate the elements and content of a web page dynamically.

JSX is an abbreviation for "JavaScript XML." It is a syntax extension for JavaScript often used with React, a popular JavaScript library for building user interfaces.
